# 🧠 NeuroScope

NeuroScope is an interactive Streamlit app that predicts a **Cognitive Score** from lifestyle habits, a live **Reaction Time Test**, and a live **Memory Test** — instead of asking users to type in numbers they'd have no way of knowing.

Built on a `RandomForestRegressor` trained with scikit-learn.

> ⚠️ **Disclaimer:** NeuroScope is a wellness/educational demo, not a medical or clinical tool. Its predictions and suggestions are for informational purposes only.

---

## ✨ Features

- **Lifestyle questionnaire** — sleep, stress, diet, screen time, exercise, caffeine
- **Live Reaction Time Test** — full-screen red → green flash, click as fast as you can
- **Live Memory Test** — memorize a 4-digit sequence, then recall it
- **Cognitive Score prediction** via a pre-trained Random Forest model
- **Results dashboard** — gauge chart, interpretation band (Excellent / Very Good / Average / Needs Improvement)
- **Personalized insights** based on the user's actual inputs
- **General cognitive-boosting tips** — posture, hand exercises, diet, stress management, sleep, activity
- **Radar chart** comparing the user's metrics against healthy reference values
- **Gold dust shower animation** on every completed step, for a bit of delight
- Dark, glassmorphism-inspired UI in a green color theme

## 📁 Project Structure

```
neuroscope/
├── app2.py                    # Main Streamlit app
├── train_model2.py            # Script to train the model from raw data
├── requirements.txt           # Python dependencies
├── model2.pkl                 # Trained RandomForestRegressor (generated)
├── gender_mapping2.pkl        # Category -> code mapping for Gender (generated)
├── diet_mapping2.pkl          # Category -> code mapping for Diet_Type (generated)
├── exercise_mapping2.pkl      # Category -> code mapping for Exercise_Frequency (generated)
└── human_cognitive_performance.csv   # Training dataset (not included — bring your own)
```

The four `.pkl` files are generated by running `train_model2.py` — they are not meant to be hand-written or edited directly.

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/<your-username>/neuroscope.git
cd neuroscope
```

### 2. Create a virtual environment (recommended)

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Train the model

Place your dataset (`human_cognitive_performance.csv`) in the project root, then run:

```bash
python train_model2.py
```

This generates `model2.pkl`, `gender_mapping2.pkl`, `diet_mapping2.pkl`, and `exercise_mapping2.pkl` in the same folder.

### 5. Run the app

```bash
streamlit run app2.py
```

The app will open automatically in your browser at `http://localhost:8501`.

## 🧩 How Reaction Time & Memory Are Measured

Rather than asking users to type these in (which they'd have no way of knowing), NeuroScope measures them live:

- **Reaction Time** — the screen flashes red, waits a random 2–5 seconds, then turns green. The time between the screen turning green and the user clicking is recorded in milliseconds.
- **Memory Test Score** — a random 4-digit sequence is shown for 5 seconds, then hidden. The user types back what they remember, and accuracy is scored 0–100 based on how many digits match by position.

Both values are automatically fed into the model alongside the lifestyle inputs.

## 📦 Deployment (Streamlit Community Cloud)

1. Push this repo to GitHub, including `model2.pkl` and the three mapping `.pkl` files (or regenerate them as part of a build step).
2. Go to [share.streamlit.io](https://share.streamlit.io/) and connect your GitHub account.
3. Select this repository and set the main file path to `app2.py`.
4. Deploy — Streamlit Cloud will install everything listed in `requirements.txt` automatically.
